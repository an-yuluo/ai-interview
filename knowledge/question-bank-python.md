## Python 核心机制

### Q1: GIL 与多线程
**问题**：Python 的 GIL（全局解释器锁）是什么？它对多线程有什么影响？
**追问链**：
- GIL 为什么存在？Python 3.13 的 no-GIL 模式了解吗？
- CPU 密集型和 IO 密集型任务分别怎么选择并发方案？
- `multiprocessing` 和 `threading` 的区别？进程间通信怎么做？
- `concurrent.futures` 的 ThreadPoolExecutor 和 ProcessPoolExecutor 怎么选？

### Q2: 装饰器与元编程
**问题**：Python 的装饰器原理是什么？手写一个带参数的装饰器。
**追问链**：
- `functools.wraps` 的作用是什么？不加会怎样？
- 类装饰器和函数装饰器有什么区别？
- 描述符（Descriptor）协议是什么？和 property 的关系？
- 元类（Metaclass）是什么？在什么场景下会用到？

### Q3: 生成器与迭代器
**问题**：生成器和普通函数有什么区别？`yield` 的底层机制是什么？
**追问链**：
- 生成器的 `send()` 方法怎么用？和 `next()` 有什么区别？
- `yield from` 的作用是什么？怎么实现协程委托？
- 迭代器协议（`__iter__` 和 `__next__`）怎么实现？
- 生成器在处理大数据时有什么优势？举个实际场景。

### Q4: MRO 与面向对象
**问题**：Python 的多重继承是怎么解决方法查找顺序的？MRO 是什么？
**追问链**：
- C3 线性化算法的原理是什么？
- `super()` 在多继承中的行为是怎样的？
- `__new__` 和 `__init__` 的区别？单例模式怎么实现？
- Python 的鸭子类型和抽象基类（ABC）各自适用什么场景？

## 评价维度

| 维度 | 优秀表现 | 不合格表现 |
|------|---------|-----------|
| 语言核心 | 深入理解 GIL、描述符、MRO 等底层机制 | 只停留在语法层面，不理解原理 |
| 异步编程 | 能设计高性能异步方案，理解事件循环 | 混淆同步异步，不会用 asyncio |
| Web 开发 | 熟悉框架原理，能处理复杂业务场景 | 只会简单 CRUD，不了解中间件 |
| 数据处理 | 有大规模数据处理经验，注重性能 | 只会基础 API，无优化意识 |
