# 数据工程师题库

> 按考察维度分类，每个问题附带追问链。面试官根据候选人技术栈选择。

---

## SQL 与数据库

### Q1: SQL 进阶查询
**问题**：写一个窗口函数，计算每个用户最近 7 天的滚动消费总额。
**追问链**：
- `ROW_NUMBER()`、`RANK()`、`DENSE_RANK()` 的区别？
- `PARTITION BY` 和 `GROUP BY` 的区别？窗口函数的执行顺序是什么？
- CTE（公用表表达式）和子查询各有什么优缺点？递归 CTE 怎么用？
- 大表关联查询慢，你怎么优化？

### Q2: 数据库设计与优化
**问题**：怎么设计一个高并发的订单表？分库分表的策略有哪些？
**追问链**：
- 水平分表和垂直分表的区别？分表键怎么选择？
- 分库分表后怎么处理跨库 JOIN 和分布式事务？
- 数据库读写分离怎么做？主从延迟怎么处理？
- 数据归档策略怎么设计？历史数据怎么管理？

## 数据仓库

### Q3: 数仓建模
**问题**：数据仓库的分层架构是什么？ODS、DWD、DWS、ADS 各层的职责？
**追问链**：
- 维度建模和范式建模的区别？星型模型和雪花模型各适用什么场景？
- 事实表有哪几种类型？（事务事实表、周期快照、累积快照）
- 缓慢变化维（SCD）有哪几种处理方式？Type 2 怎么实现？
- 你们数仓的指标体系怎么管理？怎么保证指标口径一致性？

### Q4: 数据质量与治理
**问题**：你们怎么做数据质量保障？遇到过哪些数据质量问题？
**追问链**：
- 数据质量校验规则有哪些维度？（完整性、准确性、一致性、时效性）
- 数据血缘（Data Lineage）是什么？怎么实现？
- 元数据管理怎么做？数据目录（Data Catalog）用过哪些工具？
- 数据资产的价值怎么评估？怎么做数据分级？

## 大数据技术栈

### Q5: Spark 核心
**问题**：Spark 的 RDD、DataFrame、Dataset 三者有什么区别？
**追问链**：
- Spark 的宽依赖和窄依赖是什么？Stage 是怎么划分的？
- Spark 的 Shuffle 机制是什么？Shuffle 对性能的影响？
- Spark SQL 的 Catalyst 优化器了解吗？它做了哪些优化？
- 数据倾斜（Data Skew）怎么发现？怎么解决？

### Q6: Flink 流计算
**问题**：Flink 和 Spark Streaming 的区别是什么？Flink 的时间语义有哪些？
**追问链**：
- Event Time 和 Processing Time 的区别？Watermark 的作用和原理？
- Flink 的窗口（Window）类型有哪些？滚动窗口和滑动窗口怎么配置？
- Flink 的状态管理（State）有哪几种？Checkpoint 机制的原理？
- Flink 的 Exactly-Once 语义是怎么保证的？两阶段提交协议了解吗？

## ETL 与数据管道

### Q7: ETL 流程设计
**问题**：你设计过 ETL 流程吗？ETL 的关键步骤有哪些？
**追问链**：
- 增量同步和全量同步怎么选？CDC（变更数据捕获）怎么做？
- ETL 中的数据转换（清洗、聚合、关联）怎么设计？
- 任务调度用什么工具？Airflow 的 DAG 怎么设计？
- ETL 任务失败了怎么处理？怎么保证幂等性和可重跑？

### Q8: 数据管道架构
**问题**：你们的数据管道架构是什么样的？实时和离线怎么结合？
**追问链**：
- Lambda 架构和 Kappa 架构的区别？各适用什么场景？
- 数据湖（Data Lake）和数据仓库的区别？湖仓一体了解吗？
- 消息队列（Kafka）在数据管道中的角色是什么？
- 数据管道的监控和告警怎么设计？数据延迟怎么处理？

## 评价维度

| 维度 | 优秀表现 | 不合格表现 |
|------|---------|-----------|
| SQL 能力 | 熟练写复杂查询，有分库分表经验 | 只会简单 CRUD，不会窗口函数 |
| 数仓建模 | 理解建模方法论，有实际落地经验 | 概念模糊，分不清分层职责 |
| 大数据技术 | 深入理解 Spark/Flink 原理，能调优 | 只会提交任务，不会排查问题 |
| 数据管道 | 能设计端到端数据管道，保障数据质量 | 无架构思维，只会写单个脚本 |
