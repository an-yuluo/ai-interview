# 后端开发题库

## Java 语言特性

### Q1: HashMap 底层原理
**问题**：HashMap 的底层是怎么实现的？JDK 1.8 做了什么优化？
**追问链**：
- put 操作的完整流程是什么？
- 什么时候链表会转成红黑树？阈值为什么是 8？
- HashMap 是线程安全的吗？ConcurrentHashMap 怎么保证线程安全？
- 如果自定义对象作为 key，需要重写哪些方法？

### Q2: JVM 内存模型
**问题**：JVM 的内存区域是怎么划分的？
**追问链**：
- 什么情况下会发生 StackOverflowError？
- 堆内存溢出（OOM）怎么排查？你用过哪些工具？
- 什么时候 Full GC 会触发？Full GC 频繁怎么优化？
- G1 和 ZGC 的区别是什么？

### Q3: Spring IoC 和 AOP
**问题**：Spring 的 IoC 容器是怎么工作的？
**追问链**：
- Bean 的生命周期你了解吗？
- 循环依赖怎么解决的？三级缓存的作用分别是什么？
- AOP 的底层是 JDK 动态代理还是 CGLIB？什么时候用哪个？
- Spring 事务传播机制有几种？什么情况下事务会失效？

## 项目深挖模板（通用）

1. 项目背景和业务目标是什么？
2. 你在其中扮演什么角色？团队有几个人？

5. 性能指标是多少？怎么做的性能测试？
6. 上线后遇到过什么问题？怎么排查的？
7. 如果重新做一次，你会有什么改进？
8. 这个项目的业务结果怎么样？有没有量化数据？
